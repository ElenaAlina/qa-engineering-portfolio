# Recon Methodology

## 1. Scope Review
- Read program rules
- Identify allowed assets
- Note restricted testing

## 2. Asset Discovery
- Main application
- Subdomains
- APIs
- Mobile endpoints

## 3. Technology Fingerprinting
- Web server
- Framework
- JavaScript libraries
- Authentication providers

## 4. Content Discovery
- robots.txt
- sitemap.xml
- JavaScript files
- Public documentation

## 5. Endpoint Discovery
- Parameters
- API routes
- Hidden functionality
- Admin panels

## 6. User Role Mapping
- Anonymous user
- Standard user
- Admin functionality exposure

## 7. Prioritization
Focus on:
- Authentication
- Authorization
- User management
- File uploads
- Search functionality
- API endpoints
